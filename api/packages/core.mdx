## Overview

`@morjs/core` is the runtime foundation of every MorJS mini-program. It wraps the native `App()`, `Page()`, and `Component()` constructors with a unified hook and adapter system so that plugins and cross-platform logic can intercept every lifecycle event.

| **Module formats** | CommonJS (`lib/`), ES modules (`esm/`) |

```bash
npm install @morjs/core
```

## Quick start

<CodeGroup>
```ts app.ts
import { createApp } from '@morjs/core'

createApp({
  globalData: {},
  onLaunch(options) {
    console.log('launch', options)
  }
})
```

```ts page.ts
import { aPage } from '@morjs/core'

aPage({
  data: { title: 'Hello MorJS' },
  onLoad() {
    this.setData({ title: 'Loaded' })
  }
})
```

```ts component.ts
import { aComponent } from '@morjs/core'

aComponent({
  props: { label: '' },
  data: { visible: false },
  methods: {
    show() { this.setData({ visible: true }) }
  }
})
```
</CodeGroup>
